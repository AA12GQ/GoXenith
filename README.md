## ℹ️ 项目简介

>`GoXenith`(xenith)传达顶点、巅峰的意义,有意将其打造为高性能、功能齐全的 API 框架, 将以论坛社区为主题作为服务端，基于gin, cobra, viper, zap, ent, protobuf, redis, mysql, email, jwt.

# 👷 构建
在项目根目录下运行
```
    make all
```
## 🚀 部署

```
    GOOS=linux GOARCH=amd64 make build-direct
```
